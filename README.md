# CUProf
A collection of CUDA application profiler.

## Requirements

* CUDA >= 12.0
* NVBit = 1.7.1

## Tools

* `mem_trace`: Dump memory kernel-wise access traces of CUDA applications.


* `app_metric`: Dump application information, such as memory reference counts, etc.
